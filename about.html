📘 O projekcie

Ten projekt to prosta strona internetowa prezentująca dane hydrologiczne i meteorologiczne z oficjalnego API IMGW – Instytutu Meteorologii i Gospodarki Wodnej.

Celem projektu jest prezentacja danych w nowoczesnej, responsywnej formie – z możliwością filtrowania, wyszukiwania i sortowania.

🛠️ Technologie
  • HTML5 + CSS3
  • Bootstrap 5
  • JavaScript (ES6+)
  • API: https://danepubliczne.imgw.pl/api/
  • GitHub Pages (hosting)
🔗 Źródło danych

Dane są pobierane w czasie rzeczywistym z publicznego API IMGW. Więcej informacji: https://danepubliczne.imgw.pl/api/
