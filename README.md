# 💧 Stan rzek w Polsce – projekt wdrożeniowy

Projekt front-endowy, który wykorzystuje dane z publicznego API IMGW do wizualizacji poziomu wód w Polsce. Pozwala filtrować stacje hydrologiczne wg województwa i przeglądać ich aktualne pomiary.

## 🔗 Link do strony
👉 https://Zuzanna74015.github.io/stan-rzek-w-polsce/

## 🧰 Technologie
- HTML, CSS, JS
- Bootstrap 5
- API: [IMGW Hydrologia](https://danepubliczne.imgw.pl/api/data/hydro/)

## 📄 Struktura
- `index.html` – strona startowa
- `kategorie.html` – przegląd stacji hydrologicznych
- `about.html` – o projekcie
- `main.js` – logika pobierania i filtrowania danych
